# 3.2.4 其他镜像问题

前面所描述的方法并不仅局限于单个点电荷：
任何靠近接地导电面的稳定电荷分布都可以通过引入它的镜像电荷这种方法处理 —— 因此命名为镜像法。

下面是最优美的一个。
